# Design System

Reference guide for styling standards across camdenrothleder.dev.

---

| Role | Color | Hex |

| Surface | Surface | `#E8DFC7` |

---

## Light Mode

The website maintains a **light-mode aesthetic** throughout.

- **Page background:** Cream (`#F0E9D6`)
- **Text color:** Ink (`#2A2419`)
- **Secondary surfaces:** Surface (`#E8DFC7`)
- **Muted text/borders:** Muted (`#6E6650`)
- **Accent colors:** Olive, Terracotta, Warm (use sparingly for emphasis)

---

## Usage Guidelines

- **Always check this file** before introducing new colors or styling patterns
- Extend the Tailwind config if a new token is genuinely needed, rather than using arbitrary hex values
- Maintain WCAG AA contrast ratios for all text against its background
- Keep consistency with the existing dark-mode minimalist aesthetic
